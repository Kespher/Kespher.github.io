一、早饭 🌅

（一）低价格 💰

锅边（咸鲜口）配油条、油饼、虾酥、三角糕、马耳（甜口）
⚠ 老板的手艺比较随机
晶晶有味：推荐①馒头、②发糕、③菜包 （有肉）
⚠ 福州遍地都是，但是早上会有大爷大妈抢，所以要早一点买

（二）中价格 💵

  • 麦当劳

（三）高价格 💎

  • 没吃过…

二、午饭 / 晚饭 🍽️

（一）低价格 💰

  • 沙县小吃 / 尚干小吃
  • 安平伊杰拌粉干
  • 依强牛滑
  • 达道牛滑
  • 学军盛记
  • 广达炖罐
  • 淳百味

（二）中价格 💵

店名 推荐菜品 备注
周麻婆 - -
小叫天 - -
醉得意 荔枝肉、醉排骨 -
榕树下饭店 ①炒白果（甜鲜口）
②醉排骨（咸甜口）
③酸菜鱼（黑鱼）		 地址：鼓楼区鼓楼实验小学附近，旁边有一个金城小区
注：店是比较福州口味的大排档，可以一试，我个人非常喜欢
蹄膀破店 - 适合多人吃
霞浦海鲜 - 适合多人吃
人人江记 - 适合多人吃

（三）中高价格 💎

店名 类型 地址/备注
朱富贵 小火锅，主要吃肉和海鲜 地址：我只知道东二环泰禾地下有一家，空间很大
筷子豪情 和朱富贵是同类型的店 ⚠ 这家人很多很挤，请谨慎选择
注：筷子豪情之前是大排档，后面没钱了和人合资做的小火锅
王庄阿咪 - -
回魏大排档 和王庄阿咪是同类型的店 -
同君福酒楼 - -

（四）高价格 💎💎

  • 海天盛宴：海鲜火锅自助，主要吃生食和烫海鲜
    📝 钱实在太多了可以试试这家，当然性价比不高

  • 雍和会海鲜姿造

  • 聚春园

三、小吃 🥟

（一）低价格 💰

捞化（咸鲜口）：我推荐东街口附近一个比较偏僻的店，找不到随便找一家吃也行
📝 捞化 = 高汤 + 兴化粉(一种很细的米粉) + 现捞的猪肝鸭胗之类的
  • 建瓯光饼 / 征东饼
  • 碗糕 / 茯苓糕
  • 萝卜糕 / 菜头粿

（二）中价格 💵

  • 鱼丸 / 鱼滑：推荐去永和鱼丸吃
    📝 想买回去吃推荐去连江鱼丸

  • 肉燕 / 燕丝：推荐去同利肉燕吃

（三）高价格 💎

潮福城：推荐①烤乳鸽 ②绿豆椰汁糕 ③黑椒牛仔骨
📝 这家主要吃广式茶点，勉强可以当正餐吃
  • 舒友：和潮福城是同类型的店

（四）神秘… 🤫

佛跳墙：虽然不是小吃但是建议当小吃品尝一下
📝 去聚春园一般能吃到，但是有点小贵而且外地人可能觉得味道一般
  • 坛烧八味：佛跳墙低配版，比佛跳墙便宜

四、饮料 🥤

（一）低价格 💰

  • 惠尔康全家桶：①菊花茶 ②冬瓜茶 ③优の冻 …
  • 康师傅全家桶：①冰糖雪梨 ②酸梅汤
  • 贝奇野菜

（二）中价格 💵

  • 喜茶：黑糖波波牛乳
  • 奈雪的茶：果茶系列
  • 唐沫茶兮：富裕（好像是这个音）
  • 某有茶：黑糖珍珠奶茶

（三）高价格 💎

  • 希腊奶…

五、甜品 🍰

  • 芋泥
  • 没牙伯花生汤
  • 藕糕
  • 饼夫人
  • 马蹄糕
  • 拗九粥

🎉 福州美食，等你来尝！
